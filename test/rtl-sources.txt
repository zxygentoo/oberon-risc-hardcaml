# The original Project Oberon / OberonStation Verilog is NOT vendored into this repo (its
# licensing is unclear and we prefer not to redistribute it). Instead run.sh fetches it on
# demand into test/_po/ and verifies it against the pins below — so a fresh clone with `verilator`
# installed Just Works, while we ship only these facts, never the copyrighted Verilog.

# against unknown RTL. Updating to a newer upstream revision is a deliberate edit of this file.
